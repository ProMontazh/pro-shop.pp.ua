Датчик руху Ajax DualCurtain Outdoor Jeweller типу «подвійна штора»

Бездротовий двоспрямований ІЧ-датчик руху Ajax DualCurtain Outdoor Jeweller типу «штора» призначений для одночасного захисту двох зон або двох протилежних напрямків на вулиці та в приміщенні. Обладнаний чотирма інфрачервоними сенсорами (по 2 на кожну сторону) для створення двох вузьких вертикальних променів виявлення. Кути огляду: 4,5° по горизонталі та 90° по вертикалі для кожної сторони. Дальність виявлення 4–15 метрів для кожної сторони (налаштовується окремо повзунками). Детектування в ближній зоні зменшує сліпу зону біля датчика, відхилення кута огляду на 3° для точного налаштування. Алгоритми SmartDetect та ELSA забезпечують мінімум хибних спрацювань. Імунітет до тварин заввишки до 80 см. Захист IP54, дальність зв'язку до 1700 м, автономна робота до 4 років від двох батарей CR123A. Основні характеристики:
  • Двоспрямований датчик «штора» для захисту двох зон одночасно
  • 4 інфрачервоні сенсори (по 2 на кожну сторону)

  • Висота встановлення: 0,8–1,3 м або 0,5–0,6 м від підвіконня
  • Швидкість виявлення: 0,3–2,0 м/с
  • Імунітет до тварин заввишки до 80 см
  • 3 рівні чутливості (налаштовується)

  • Система антимаскування

Двоспрямований датчик — для чого потрібен:
  • Два незалежні промені виявлення на протилежних сторонах датчика
  • Захист двох зон одночасно одним датчиком
  • Економія на кількості датчиків та батарей
  • Контроль проходу з обох боків (двері, коридор, арка)
  • Захист двох вікон кутової кімнати одночасно
  • Окреме налаштування дальності для кожної сторони
Чотири ІЧ сенсори (2+2):
  • По 2 сенсори на кожну сторону для підвищення точності
  • Спрацювання тільки при підтвердженні обома сенсорами сторони
  • Різко зменшує ймовірність хибних тривог
  • Незалежна робота кожної сторони
Вузький промінь (4,5° × 90°):
  • Горизонтальний кут: 4,5° — ще вужчий ніж у Curtain Outdoor (8°)
  • Вертикальний кут: 90° — покриває всю висоту від підлоги до стелі
  • Мінімум хибних спрацювань від руху в стороні
  • Ідеально для вузьких проходів, дверей, вікон
Детектування в ближній зоні:
  • Унікальна функція для зменшення сліпої зони під датчиком
  • Верхній ІЧ сенсор отримує додатковий сектор огляду (нахил 40° вниз)
  • Фіксує рух близько до самого датчика
  • Увімкнення окремо для кожної сторони повзунками
  • Виявляє спроби обійти датчик знизу
Відхилення кута огляду на 3°:
  • Можливість відхилити сектор огляду горизонтально на 3°
  • Налаштовується окремо для кожної сторони повзунками
  • Точне наведення променя на зону охорони
  • Корисно при нестандартному розташуванні вікон/дверей
Налаштування дальності (4–15 метрів для кожної сторони):
